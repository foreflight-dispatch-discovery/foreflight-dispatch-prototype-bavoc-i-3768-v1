# BAVOC-I-3768 v1 — Air Refueling Mission Planner Prototype

This is a single-page clickable prototype for the ForeFlight Dispatch Air-to-Air Refueling mission planning module, built from seven discovery sessions with military aviation operators (European and US air forces). It demonstrates a mission-first planning approach where tanker and receiver aircraft are coordinated within a named mission object — rather than tagged onto individual flight plans after the fact — with explicit support for Primary and Contingency courses of action. The prototype runs entirely in the browser with no dependencies.
